## 个人版与团队版规则

个人版和团队版共享同一套底层计费与算力扣费体系：

- **个人版**：适合单人独立使用，所有模型调用和 Agent 服务消耗均从该订阅账号中扣除。
- **团队版**：适合多人多角色协作使用，团队成员共享同一个主扣费账号。团队内所有成员产生的模型调用、任务执行和平台资源消耗，都会统一从该主账号中扣除点数。

默认最多同时部署 5 个常驻 Agent，具体数量随套餐可能不同。

## Agent 计费与扣费机制

Agent 平台采用双重消耗计费模式：

1. **平台资源消耗**：Agent 运行、状态机维持、动态上下文编排等平台基础设施资源，按部署类型以固定点数/小时费率扣除对应算力点——Swarm（蜂群）部署按 **150 点/小时** 计费，Sub（单 Agent）部署按 **10 点/小时** 计费。
2. **模型调用消耗**：Agent 运行过程中向上游模型网关（CodeGW）发起的 Token 请求，将按各模型的实际消耗折算点数，直接从主账号中实时扣除。

| 消耗类型 | 扣费路径 |
|---|---|
| Agent 平台资源消耗 | Swarm 150 点/小时，Sub 10 点/小时 |
| Agent 调用模型产生的费用 | 从主扣费账号余额/点数中实时扣除 |
