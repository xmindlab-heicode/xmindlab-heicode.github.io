# Team Usage

<Callout type="warn" title="Current Status Notice">
  Organization, member invitations, role assignment, and other team-collaboration capabilities are still in the planning/backend stage, and a dedicated team management interface has not yet been made available. In the content below, the parts marked as "implemented" refer to mechanisms — such as approvals, secret custody, and auditing — that already run in real, single-task/single-Agent scenarios. Parts involving multi-member organizations, the team role system, and team usage/governance dashboards are currently planned capabilities that cannot yet be used directly through any interface.
</Callout>

This section describes how teams collaborate when using Heicode, including quickstart, task collaboration workflow, roles and permissions, approval/delivery/release, team governance, and handling common issues.
