# 团队使用

<Callout type="warn" title="当前状态说明">
  团队协作中的组织、成员邀请、角色分配等能力目前仍在规划/后端阶段，尚未对外开放独立的团队管理界面。以下内容中标注为"已实现"的部分，指的是审批、密钥托管和审计等已经在单任务/单 Agent 场景下真实运行的机制；涉及多成员组织、团队角色体系、团队用量与治理看板的部分目前属于规划中的能力，尚不能通过任何界面直接使用。
</Callout>

本节介绍团队在使用 Heicode 时的协作方式，包括快速开始、任务协作流程、角色和权限、审批与发布、团队治理，以及常见问题处理。
