# SEO Canonical Policy

## Canonical domain

The only public documentation domain is:

`https://docs.heicode.cc`

## Localization

English and Chinese pages should represent the same product information.

Requirements:

- Keep language navigation consistent.
- Avoid duplicate pages with conflicting content.
- Add canonical metadata through the documentation framework.
- Add hreflang metadata when both language versions exist.

## Publishing rules

Do not publish internal-only pages or preview features as generally available.
