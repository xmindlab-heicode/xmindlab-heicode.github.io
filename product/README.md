# Heicode 产品说明目录

本目录只放面向 Heicode 产品、用户、团队和商业使用的正式说明。

适合放在这里的内容：

- 产品介绍与使用指南。
- 团队使用、权限、审批、交付和发布说明。
- Agnet 构建与工作模式说明。
- 计费、套餐、安全、凭证、隐私和法律说明。
- 产品版本更新。

不适合放在这里的内容：

- 项目书、专项分析、内部实施计划。
- 某个仓库的本地工程规则。
- 临时交付记录、会议记录、开发状态报告。

这些内容应放到 `projects/`、对应业务仓库，或后续按主题新增目录。
