# 个人及团队计费说明

平台采用 `X` 作为统一计费单位。当前预计：

`1X = 标准基础模型连续工作额度（约等于10美金）`

最终价格以实际购买页面或合同约定为准。

## 套餐类型

| 套餐类型 | 金额档位 | 服务倍率 | 主要能力 |
|---|---:|---:|---|
| 基础版 | 14 / 20 / 120 美元 | 5X | 以 Qwen 系列等基础模型为主，支持连续性工作 |
| 高级版 | 600 / 800 / 1000 美元 | 20X | 以 GPT-5 系列作为基础模型，支持更高质量的连续性工作 |

## 基础版说明

基础版金额档位为 `14美元`、`20美元`、`120 美元`。

该版本提供 `5X` 服务能力，主要以 Qwen 系列等基础模型作为连续性工作的基础模型。

实际使用过程中会按模型调用消耗扣费；国内模型和海外模型，例如 Gemini、Claude、GPT 等，可作为补充能力提供有限服务，并按实际使用量扣费。

## 高级版说明

高级版金额档位为 `600 美元`、`800 美元`、`1000 美元`。

该版本提供 `20X` 服务能力，以 GPT-5 系列作为连续性工作的基础模型，适合更复杂、更高质量的使用场景。

国内模型及海外模型，例如 Gemini、Claude、GPT 等，也会按照实际调用情况扣费。

## 个人版与团队版

个人版和团队版均使用同一个扣费账号进行统一扣费。

个人版适合单人使用，所有模型调用和服务消耗均从该扣费账号中扣除。个人版最多可部署 `5` 个 Agent。

团队版适合多人协作使用，团队成员共享同一个扣费账号。团队内所有成员产生的模型调用、任务执行和相关服务消耗，都会统一从该扣费账号中扣除。团队版最多可部署 `8` 个 Agent。

## Agent 计费说明

Agent 平台按照实际使用的 `EU` 进行扣分。

Agent 运行过程中产生的平台资源消耗，会按实际使用量扣除对应 `EU`。

如果 Agent 调用了模型，该模型调用产生的费用也会从主扣费账号中扣除。

也就是说，Agent 使用会产生两类消耗：

| 消耗类型 | 扣费方式 |
|---|---|
| Agent 平台资源消耗 | 按实际使用 `EU` 扣分 |
| Agent 调用模型产生的费用 | 从主扣费账号中扣除 |
