# 企业版计费说明

企业版主要采用**按席位购买（Per-Seat）**的计费模式，专为企业级团队、跨部门协作、高安全审计以及多人复杂研发场景量身定制。

企业版标准席位默认 `5` 个起售，提供更高级别的权限防线、全链路审计记录以及集中的账单管理。

## 企业版套餐与计费规则

企业版根据组织规模和技术深度，划分为以下三个档位标准：

| 套餐类型 | 人民币定价 (人/月) | 美元定价 (人/月) | 起售门槛 | 核心权益及说明 |
|---|---|---|---|---|
| **企业 标准版** | ￥699 / 人/月 <br>(年付 ￥6,990) | $99 / 人/月 <br>(年付 $990) | `5` 席位起售 <br>(月付 ￥3,495 起) | 包含高级研发协同、基础资源隔离、企业级权限防火墙及全量日志审计。 |

| **企业 定制版** | 商务洽谈 | 商务洽谈 | `50` 席位以上 | 支持混合云/私有化部署、完全隔离的凭证中心、定制化团队 MCP 及专属模型通道接入。 |

## 企业版超额计费机制

企业版各席位均包含高规格的基础算力包与服务额度。当实际团队用量超出套餐包含的默认范围后，超出部分将按下述口径进行实际消耗计费：

- **模型调用消耗**：统一经由企业级模型网关 CodeGW 进行细粒度分账审计，超出基础额度后按实际 Token 吞吐阶梯计费。
- **Agent 资源消耗**：企业内部署的 Sub Agent 或 Swarm 蜂群在执行大规模任务图时产生的计算资源，按实际使用的 `EU` 进行动态扣减。

## BYOK (Bring Your Own Key) 说明

企业版全面支持 **BYOK** 模式。

企业客户可以无缝接入自有且受控的模型服务 Key（例如自有的 OpenAI、Anthropic Claude、Google Gemini 或企业级专属模型 API）。

- **BYOK 费用归属**：启用 BYOK 时，底层模型调用产生的 Token 费用由企业自有的服务商账号直接承担，降低平台中转成本。
- **平台管理费用**：即便使用企业自有 Key，平台侧提供的 Agent 架构调度、资源准备向导（Resource Grant）、团队高危操作审批防火墙、全链路审计以及管理驾驶舱等核心功能，仍将根据企业版席位及平台规则正常计费或扣除基础平台资源。
