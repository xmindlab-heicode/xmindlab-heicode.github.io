## 企业版说明 企业版主要按照席位进行购买，适合企业级团队和多人协作场景。 企业版席位默认 `3` 个起售。 每个席位默认价格为 `1000 美元`，并按照 `20X` 标准提供服务能力。 每个席位最多可部署 `8` 个 Agnet。 企业版支持更高规格的团队协作、权限管理、资源分配和统一账单管理。 ## 企业版计费规则 | 项目 | 说明 | |---|---| | 计费方式 | 按席位购买 | | 起售数量 | 默认 `3` 个席位起售 | | 单席位价格 | `1000 美元 / 席位` | | 服务倍率 | `20X` | | Agnet 数量 | 每个席位最多 `8` 个 | | 超出部分 | 按实际消耗额外计费 | | BYOK | 支持 BYOK，可使用客户自有模型 Key | ## 企业版超额计费 企业版每个席位按照 `20X` 标准提供基础服务能力。 当实际使用量超出套餐包含的服务范围或额度后，超出部分会按照实际消耗进行计费。 超额消耗包括但不限于： | 消耗类型 | 计费方式 | |---|---| | 模型调用消耗 | 按实际调用量计费 | | Agnet 平台资源消耗 | 按实际使用 `EU` 扣分或计费 | | 超出默认服务范围的资源使用 | 按实际消耗计费 | ## BYOK 说明 企业版支持 `BYOK`。 BYOK 即 `Bring Your Own Key`，客户可以接入自己的模型服务 Key，例如自有的 OpenAI、Claude、Gemini 或其他模型服务 Key。 使用 BYOK 时，模型服务侧产生的费用由客户自有 Key 对应的服务商账号承担；平台侧产生的 Agnet、调度、协作和管理等资源消耗，仍可能按照平台规则进行计费。
